# Ising Model 

## General Chain-Structured Models

There is an important probability distribution used in applications (Liu, 2008):

$$
\pi(\mathbf x) \approx \exp\Big\{-\sum_{i=1}^dh_i(x_{i-1},x_i)\Big\} 
$$

where $$\mathbf x = (x_0,x_1,\ldots,x_d)$$.
